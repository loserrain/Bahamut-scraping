# Bahamut-scraping
用Scrapy去爬巴哈姆特場外休憩的標題跟預覽內文，並輸出csv檔<br />
可以設定要爬多少頁<br />
不過過度對巴哈伺服器發出請求會對巴哈伺服器造成影響<br />
還在測試的程式碼，建議不要一次設定爬太多頁<br />
